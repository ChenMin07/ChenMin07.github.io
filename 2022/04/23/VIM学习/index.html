VIM的使用

VIM快捷键

删除单词

  • dw 删除当前单词至后一个单词的开头
  • de 删除当前单词至末尾
  • daw 删除光标所在的单词至后一个单词的开头
  • D 删除至光标至本行末尾的
  • d0 删除至本行开头
  • dd删除一行

撤销

  • u
  • ctrl + r反撤销

函数跳转

移动光标

Vim 中如何快速移动光标？ | Harttle Land

  • $是移动到行末尾
  • 0是移动到行首
  • shift + ]是移动到下一个函数
  • shift + [是移动到上一个函数
  • ctrl + d是前进半页，+ u是后退半页
  • w是移动到下一个单词的词首
  • b是上一个单词的词首
  • e是下一个单词的结尾

剪切、删除并插入

Vim 进阶 - 剪切（删除）、复制、粘贴与寄存器 - SegmentFault 思否

插入

  • 在光标前 i， 行前I
  • 光标后 a， 行后A
  • 光标的上一行插入O，下一行插入o

显示当前编辑文件名

  • ctrl + g

bash下：

  • alt + d是删除右边
  • ctrl + w是删除左边
